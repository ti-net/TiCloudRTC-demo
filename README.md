# TiCloudRTC-demo

## 简介

此仓库包含基于天润RTC-SDK的示例项目。

## 目录结构说明

| 目录        | 说明                                   |
| ----------- | -------------------------------------- |
| RTC-Android | demo for Android[RTC-Android](RTC-Android) |
| RTC-Web     | demo for web[RTC-Web](RTC-Web)         |
| RTC-WeChat  | demo for web[RTC-WeChat](RTC-WeChat)   |
| Tools/      | Tools for RTC[Tools](Tools)            |
